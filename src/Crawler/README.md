# 爬虫项目

## 哔哩哔哩视频爬取 ##

> bilibili_vidio

* 爬取哔哩哔哩视频

  ```shell
  $ python bilibili.py
  ```

* 视频转音频

  ```shell
  $ python mp4towav.py
  ```

* 识别音频中的文字

  ```shell
  $ python audio_process.py
  ```
